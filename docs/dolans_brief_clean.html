核心话题

1. LLM Agent: 从"万能解药"到"昂贵玩具"

过去一年，所有人都在说 Agent 是 AI 的下一个范式。但当我们拨开营销的迷雾，所谓的 "Agent" 不过是更复杂的 prompt 模板 + RAG。

本周 arxiv 上的论文《Budget-Sensitive Discovery Scoring》用实验证明：传统方法在药物发现任务上吊打所有 LLM 配置。讽刺吗？但这就是现实。

Dolan 说：别被 "Agent" 这个词绑架了。真正的突破是推理能力的提升，而不是 prompt 技巧的堆砌。

---

2. CVPR 2026: 伪繁荣

CVPR 论文产量再创新高，但仔细看都是些什么？

- SPARROW: 视频理解论文声称 SOTA，但所谓的 "spatial precision" 连物体边界都画不准 - Model Stitching: 声称 vision model 可以任意拼接，结果连通用表示都找不到 - CNN 在癌症病理分析中的应用：学的是数据分布偏见，不是病灶

Dolan 说：CVPR 越来越像CES——展出很多，落地很少。

---

3. AI 创业泡沫: 都在裸泳

- ByteDance 暂停 Seedance 2.0：做出来的东西连自己都不敢发 - AI 心理诉讼案：AI 最擅长的两件事 1)让人失业 2)让人发疯 - "AI Wrappers" 融资潮：90% 不过是把 API 套个壳

Dolan 说：只有潮水退去，才知道谁在裸泳。
