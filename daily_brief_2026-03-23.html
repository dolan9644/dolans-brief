Agentic Investing 资管新品类：GIM 获千万天使轮，五源 + Monolith + 港大教授的豪华阵容想干什么

背景溯源：资产管理行业经历了三代更迭：主观投资人（巴菲特）→ 量化投资（文艺复兴）→ Agentic Investing。GIM（Grace Investment Machine）刚刚完成数千万元天使轮，五源资本与 Monolith 联合注资，创始人徐嘉浩是前 Neumann Advisors / 五源资本资深投资人，联创刘琦博士是港大计算机系助理教授，曾在 DeepMind 和 FAIR 任职。这套阵容说明什么？说明传统 VC 开始用一级市场眼光押注 AI Native 的资产管理逻辑。

GIM 的核心逻辑是基于自研金融时序大模型，构建多智能体投资平台。私募证券投资基金布局推进中，计划推出智能体驱动基金策略。这是典型的「第三代资管路径」—— 用 AI 取代（或者至少大幅辅助）人类投资经理的决策流程。

多智能体架构在真实资管场景的可行性仍然存疑：从研究辅助到全自动交易之间存在巨大鸿沟，尤其在合规与风控层面。一个量化策略在模拟盘上跑赢大盘，和把这个策略用到真实基金产品上，是完全不同的两件事。监管机构对 AI 全自动交易的接受度、金融市场对 AI 驱动的流动性冲击、极端行情下智能体的风控能力—— 这些都是 GIM 必须回答的问题。

从技术角度看，自进化智能体能否真正实现持续超额收益，是判断这个方向能否成立的核心指标。目前看来，GIM 的护城河更多依赖创始人徐嘉浩在一级市场的人脉和资管经验，而非纯粹的技术壁垒。五源资本和 Monolith 的入局，本质上是对「AI + 投资」赛道的一次战略卡位，而非对某个具体技术的押注。

Dolan's 锐评
五源和 Monolith 投的不是 GIM 这个团队，是「AI 取代对冲基金」这个故事的期权费。真正的问题不是智能体能不能赚钱，而是当所有资管公司都用智能体的时候，超额收益会像打新股一样消失。Agentic Investing 的本质是让镰刀更锋利，不是让韭菜翻身。

Simon Willison 的 AI 编程三分法：一场关于「代码安全」的路线战争正在爆发

背景溯源：Simon Willison 在 Pragmatic Summit 上扔出了一颗炸弹：他将 AI 编程分为三个阶段—— ChatGPT 问答（Copilot 模式）→ 代理写代码（Agent 模式）→「无人读代码」（Fully Autonomous 模式）。这个框架的重要性在于，它把整个行业正在发生的事做了一个清晰的地理标注：大多数人还停留在第一阶段，极客圈已经卷到第二阶段，而安全公司 StrongDM 则宣称要直接跳到第三阶段，声称「没人写代码、没人读代码」。

Simon Willison 的原话是：这种做法「明显不负责任」。

他的核心论点是：安全公司如果把安全责任外包给 AI，一旦失误代价是灾难性的。StrongDM 的逻辑是代码审计 AI 可以比人类更彻底，但这个逻辑忽略了一个根本问题：谁为 AI 审计失败负责？在合规领域，责任链条的断裂是致命的。

Willison 同时展示了一个用 Claude 编写任务管理 Starlette 1.0 应用并完整测试的案例。他的方法论核心是 TDD for agents：uv run pytest，测试 → 红 → 绿 → 重构循环。这套流程的本质是什么？是把人类的判断力集中在「定义问题」和「验证答案」两个环节，把「解决问题」的过程完全交给 AI。

关键洞察：AI 编程工具的真正瓶颈不是代码生成质量，而是测试覆盖率与可观测性。代码质量是由「测试质量 × 生成质量」决定的，而非单独一项。这句话戳破了很多 AI 编程工具营销的泡沫—— Copilot 可以生成代码，但如果你的测试覆盖率不够，生成的代码依然是定时炸弹。

另外值得关注的细节：Willison 用 uv run pytest 这个命令来运行测试，uv 是 Astral 旗下的包管理器，月下载量超过 1.26 亿次。这个数字意味着什么？意味着 Python 生态已经把 uv 作为事实标准。而 Astral 刚刚被 OpenAI 收购—— 这意味着 OpenAI 现在手里捏着 Python 生态最核心的入口之一。

Simon Willison 的观点链接：simonwillison.net/2026
Dolan's 锐评
StrongDM 的「无人读代码」等于宣布「我们决定把责任归零」。这不是技术革命，这是风险转移。当一家安全公司开始外包责任的时候，它服务的客户实际上在用更高的风险换更低的价格。Simon Willison 的 TDD for agents 才是真正的路—— 把人类智慧放在定义问题和验证结果上，让 AI 做执行，这是人机协作的正确姿势。

GPT-5.4 nano 降至 $0.20/M 输入：小模型价格战彻底撕裂利润底线

核心事件：OpenAI 发布 GPT-5.4 nano，$0.20/M 输入、$1.25/M 输出，价格低于 Gemini 3.1 Flash-Lite。7.6 万张照片批量描述成本约 $52.44。同期 MiniMax 2.7 发布，以 1/3 成本达到与 Z.ai GLM-5 同等的 SOTA 分数。

关键细节：Artificial Analysis 的评测显示 MiniMax 2.7 引入「自我进化」概念，声称可覆盖 30%-50% 工作流。GPT-5.4 nano 的价格战让每百万 token 的边际成本接近一碗方便面的价格。

Dolan's 锐评
小模型价格战的尽头是推理边际成本趋近于零。当使用成本可以忽略不计，基于用量的商业模式将面临根本性重构。MiniMax 的「自我进化」声明与真正的自主进化之间存在巨大差距，更接近于「上下文学习能力的工程化」—— 是个营销词汇，不是个技术突破。

OpenAI 收购 Astral（uv/ruff/ty）：AI 实验室「买开发者生态」战争升级

核心事件：Astral（uv 包管理器、ruff linter、ty type checker）被 OpenAI 收购，团队并入 Codex 团队。uv 月下载量超 1.26 亿次。

关键细节：同期 Anthropic 收购 Bun（JavaScript 运行时），Google 收购 Antigravity 团队（Cursor 基础）。Simon Willison 指出「产品 + 人才」收购随时可演变为纯人才收割。Astral 的 A 轮（Accel）和 B 轮（a16z）均为首次曝光。

Dolan's 锐评
Python 生态的控制权争夺本质上是「AI 编程工作流入口」之争—— 控制 uv 的解析器和 ruff 的代码分析能力，等于掌握了 AI 编程的事实标准。OpenAI 买走的不只是工具，是整个 Python 开发者生态的信任。Armin Ronacher 说开源被 VC 收购后的最坏情况是「fork and move on」，但这次 Astral 用的是 permissive license，最坏情况可控。好消息是代码不会消失；坏消息是维护代码的人可以消失。

Anthropic 收购 Bun：JS 运行时争夺战的背后是端侧 AI 入口卡位

核心事件：Anthropic 收购 Bun（JavaScript/TypeScript 运行时），与 OpenAI 收购 Astral 几乎同期。

关键细节：Bun 由 Jarred Sumner 开发，是 Node.js 的替代方案，主打高性能。Anthropic 的意图是将 Bun 集成到 Claude 的工具链中，加速端侧 AI 应用开发。

Dolan's 锐评
JS 运行时是 Web 端侧 AI 的入口。谁控制了 JS 引擎，谁就能在浏览器端的 AI 应用分发上占据有利位置。Anthropic 和 OpenAI 同时在「买基础设施」，说明大厂对 AI Native 开发工具的重视程度已经超过了很多人的认知。这不是人才收购，是战略锁喉。

Google 收购 Antigravity 团队：Cursor 基础团队花落谁家

核心事件：Google 收购了 Cursor 基础技术团队 Antigravity。

关键细节：Cursor 是目前最火的 AI 编程 IDE 之一，被认为是 GitHub Copilot 的最大竞争对手。Google 的意图是将 Antigravity 的技术整合到 Gemini Code 工具链中。

Dolan's 锐评
Google 买 Cursor 团队的理由和 OpenAI 买 Astral 的理由一模一样—— 争夺 AI 编程入口。问题是，Cursor 之所以强，是因为他们的团队是独立的；被 Google 收购后，还能保持同样的创新速度吗？历史答案大概率是「不能」。

GPT-5.4 mini 发布：OpenAI 的「薄利多销」策略正在建立新的价格锚点

核心事件：OpenAI 同步发布 GPT-5.4 mini，在价格和性能之间找到了新的平衡点。

关键细节：GPT-5.4 系列是 OpenAI 首次在同一次发布会上同时推出大杯、超大杯和 mini 版本，形成了完整的产品矩阵。

Dolan's 锐评
OpenAI 正在用 iPhone 的产品策略做 AI —— 高配树品牌，标配走量，mini 收割长尾。这个策略在 SaaS 行业已经验证过了，现在把它复刻到 AI 领域，本质上是把竞争焦点从「谁更聪明」转移到「谁更便宜」—— 这对整个行业来说不是好消息。

Claude Cowork 迟到两年入场：Anthropic 的桌面 AI 战略是亡羊补牢还是后发制人

核心事件：Anthropic 推出 Claude Cowork 桌面应用，由前 Slack 桌面版 / Electron 维护者 Felix Rieseberg 领导。

关键细节：Cowork 起源于「Claude Code 用户用它做非编程工作」的意外发现，人类团队仅一周半协调多个 Claude Code 实例完成产品—— AI 自己写了产品。Jensen Huang 在 GTC 上公开称赞 OpenClaw 安全模式，被认为是对 OpenClaw 的回应，但 Cowork 晚了两年才真正推出。

Felix Rieseberg 前 Slack 桌面版 / Electron 维护者，Cowork 项目负责人
Dolan's 锐评
Electron 架构的沙箱安全边界存疑—— 本地拥有完整文件系统访问权限的 AI 代理，其攻击面远超云端。Cowork「自己写了自己」的故事是强大的营销叙事，但也揭示了 AI 应用开发的根本转变：人类角色从「建造者」变为「协调者」，这个转变比大多数从业者承认的更快。

MiniMax 2.7 发布：国产大模型首次在性价比上实现全面领先

核心事件：MiniMax 2.7 发布，Artificial Analysis 显示以 1/3 成本达到与 Z.ai GLM-5 同等的 SOTA 分数。

关键细节：「自我进化」概念被引入，声称可覆盖 30%-50% 工作流。这是国产大模型首次在性价比维度上领先全球。

Dolan's 锐评
MiniMax 的「自我进化」是精心设计的市场语言。真正的自主进化需要模型在部署后持续更新权重，这和「利用上下文窗口做动态学习」是完全不同的两件事。国产大模型在工程化能力上正在快速追赶，但在基础创新上仍然有差距。这个差距不是靠价格战能弥补的。

Astral quiet Series A/B 曝光：开源 VC 融资的隐秘游戏

核心事件：Astral 的 A 轮（Accel/Casey Aylward）和 B 轮（a16z/Jennifer Li）均为首次曝光，此前仅公布种子轮。

关键细节：从 2023 年创立到 2025 年，Astral 已完成至少三轮融资，但此前完全保密。pyxis（私有 PyPI 注册表）未出现在收购公告中是重要信号。

Dolan's 锐评
开源 VC 融资的秘密化是双刃剑：社区在不知情时失去参与机会，但过早曝光可能导致竞争收购—— 这笔交易的条款可能比大多数开源收购更保护社区。但 pyxis（私有 PyPI 注册表）的缺席意味着 OpenAI 对 Astral 的定位是「工具」而非「平台」，这是一个被市场忽略的重大信息。

AI 漫剧爆发：Seedance 2.0 让分镜导演集体失业

核心事件：AI 漫剧赛道规模超 200 亿元，酱油动漫月收入 5000 万，字节红果漫剧 DAU 突破 1000 万仅用三个多月。Seedance 2.0（2026 年 2 月）让 8-10 人/部配置压缩至 3 人。

关键细节：Seedance 2.0 让分镜导演被裁撤，AI 仿真人剧成新方向。平台争夺：字节、腾讯、百度、快手全部入场，各家年框协议争夺 API 接口。

Dolan's 锐评
最先进的大模型技术（Seedance 2.0）被用于最廉价的劳动密集型内容生产—— 技术平权没有带来创作平权，而是将「创意劳动」降格为「抽卡操作」。AI 漫剧的技术迭代速度（三个月换一代）远超传统影视，任何产能壁垒都是暂时的，最终竞争回到内容本身—— 但平台已用速度建立起先发优势。

轻舟智航获 1 亿美元 D 轮：世界模型 + 强化学习是自动驾驶的下一站

核心事件：轻舟智航完成 D 轮 1 亿美元融资，投资方为国内头部主机厂、汽车电子零部件公司等产业资本。

关键细节：资金用于世界模型 + 强化学习的物理 AI 技术研发。轻舟已实现 L2++ 乘风系统 100 万台交付，与理想、奇瑞、广汽、吉利等近 10 家车企合作。2027 年目标：NOA 量产 300 万台，Robovan 十万台级别量产，Robotaxi 规模化。

Dolan's 锐评
世界模型是自动驾驶从「感知-规划分离」走向「端到端物理理解」的关键路径，但真实物理规律的建模复杂度远超视觉模型。百万量级交付数据闭环的价值被低估—— L2 与 L4 数据共享的技术底座，使得轻舟比纯 L4 公司拥有更可持续的数据来源。这个故事能不能兑现，2027 年见分晓。

食气生化获超亿元融资：碳利用颠覆石油基化工逻辑

核心事件：食气生化获上海科创知识产权基金、天创资本超亿元投资，食气梭菌发酵法以工业废气为原料生产丁醇。

关键细节：五万吨级示范项目已备案赋码，传统石油-丙烯工艺原料成本占比超 60%，生物法原料成本占比远低于此。丁醇生产每吨可减少约 4 吨 CO₂ 排放。

Dolan's 锐评
「用高技术做低附加值大宗产品」的战略选择是明智的—— 丁醇市场足够大、竞价充分，但技术壁垒与工程化放大能力才是真正的护城河。碳排放管理从「管能源」向「管碳」转变的政策拐点已至，生物制造正在将碳排放从负资产变为生产原料。这个方向值得关注，但工业放大才是真正的死亡之谷。
Dolan's 锐评：巨头绞肉机板块总结
三条主线正在同时撕裂 AI 行业的利润结构：第一，小模型价格战让推理边际成本趋近于零，基于用量的商业模式面临重构；第二，大厂争抢开源基础设施（uv、Bun、Cursor 团队）是在争夺 AI Native 开发工具的入口控制权，这个博弈和当年 IBM/微软/Google 争夺操作系统控制权如出一辙；第三，AI 应用层的创新正在从「更聪明」转向「更便宜」，这是所有技术成熟后的共同宿命，但不是技术创新者应该追求的方向。当战场转向价格，所有玩家的毛利率都会趋向社会平均劳动回报率。

Dolan's 致命锐评
本周情报指向一个残酷的现实：AI 行业正在从「技术竞争」转向「成本竞争」。OpenAI 买走 Astral、Google 买走 Cursor 团队、Anthropic 买走 Bun—— 三大实验室同时在争夺开发者生态的「基础设施」控制权，这和当年 IBM/微软/Google 争夺操作系统控制权的剧本如出一辙。但这次有一个根本性的不同：开源生态的存在使得「控制」变得更加困难。uv 可以被 fork，ruff 可以被分叉，任何 permissive license 下的代码都可以被社区继承。

真正的终局问题不是谁控制了工具，而是谁能在「工具」之上建立真正不可复制的壁垒—— 对于 AI 公司来说，这个壁垒是数据；对于资管公司来说，这个壁垒是阿尔法；对于创业者来说，这个壁垒是对某个垂直场景的深度理解。大模型的商品化会让真正有壁垒的业务浮出水面，也会让没有壁垒的「AI 公司」原形毕露。2026 年会是一个分水岭：能活下来的不是最强的 AI 公司，而是最懂某个行业「行活」的 AI 应用公司。
